BUG REPORTS

1. Книга с отрицательным значением года выпуска
    Description: - Метод POST дает возможность добавить в базу книгу с отрицательным числовым значением
    Steps to reproduce: 1) Отправить POST запрос http://localhost:5000/api/books
                           c телом, в котором заполнены поля "name" и "year":
                                {
                                   "author": null,
                                   "id": 0,
                                   "isElectronicBook": false,
                                   "name": "Random book",
                                   "year": -1999
                                }
                         2) Проверить код состояния
                         3) Проверить тело ответа
    Expected Result: - Статус ответа 400
                     - В теле ответа содержится ошибка
    Actual result: - Статус ответа 201
                   - Книга добавлена в базу
